# dataset

## 简介

该仓库集中收录常见图像去噪/超分辨率基准数据集，方便统一管理与调试。当前包含的公共数据集包括 **BSD68、BSD100、BSD500、FMD** 以及 **Set12**，总容量约 7GB，文件全部托管在 Git LFS 中。

## 目录结构

```
.
├── BSD68
├── BSD100
├── BSD500
├── FDM
└── Set12
```

- **BSD68**：由 68 张 512×512 的灰度自然图像组成，是经典的图像去噪测试集，常用于对比不同噪声强度下的 PSNR/SSIM。
- **BSD100**：Berkeley Segmentation Dataset 的 100 张子集，分辨率不一但保留丰富纹理细节，常用于图像超分辨与去噪任务的综合评测。
- **BSD500**：包含 500 张彩色自然图像，同时提供精细的人为分割标注，可用于边缘检测、语义分割或多任务联合训练。
- **FMD**：荧光显微镜去噪 (FMD) 数据集专用于泊松-高斯去噪。该数据集由 12,000 张真实荧光显微镜图像组成，这些图像是使用商业共聚焦、双光子和广角显微镜以及细胞、斑马鱼和小鼠脑组织等代表性生物样本获得的。图像平均用于有效地获得地面实况图像和 60,000 个具有不同噪声水平的噪声图像。
- **Set12**：12 张广泛使用的经典测试图（如 Lena、Barbara、House 等），图像尺寸较小但纹理多样，常用于快速验证算法效果和可视化展示。
